A Ruby client library for the [Ithos][ithos] directory server.

[ithos]: https://github.com/cryptosphere/ithos

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ithos'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ithos
